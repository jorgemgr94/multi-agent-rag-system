# Fintech Industry Playbook

## Industry Overview
Fintech is our second-largest vertical. These companies move fast, have sophisticated technical teams, and require robust security and API capabilities.

## Market Size
- **TAM:** $200B fintech software market
- **Our SAM:** $12B (sales and operations)
- **Growth Rate:** 18% CAGR

## Key Buyer Personas

### VP of Sales / CRO
- **Priorities:** Revenue growth, team efficiency, forecasting
- **Concerns:** Data accuracy, speed, scalability
- **Decision Role:** Budget owner, champion

### CISO / Security Team
- **Priorities:** SOC 2, penetration testing, data security
- **Concerns:** Third-party risk, data residency, encryption
- **Decision Role:** Security approval (often gate)

### CTO / Engineering Lead
- **Priorities:** API quality, integrations, developer experience
- **Concerns:** Technical debt, vendor lock-in, scalability
- **Decision Role:** Technical validation

## Security Requirements

### Table Stakes
1. SOC 2 Type II certification
2. Penetration testing (annual minimum)
3. Data encryption (at rest and in transit)
4. SSO/SAML support
5. Audit logging

### Nice to Have
- ISO 27001 certification
- Bug bounty program
- Data residency options (EU, etc.)

## Sales Motion

### Discovery Questions
1. "What's your current security review process for new vendors?"
2. "Do you have specific data residency requirements?"
3. "How important is API access for your integrations?"
4. "Who handles vendor security assessments?"

### Proof Points
- Globex Financial case study ($180K, 90-day enterprise close)
- SOC 2 Type II audit report
- API documentation and developer portal
- Security whitepaper

### Demo Focus
Lead with:
1. API capabilities and webhook support
2. Security features and audit logs
3. Scalability (show enterprise customer metrics)
4. Integration marketplace

## Competitive Positioning
- **vs Alpha Corp:** They only have SOC 2 Type I. Our Type II is more rigorous.
- **vs Beta Systems:** We're faster to implement and more API-friendly.

## Pricing Guidance
Fintech buyers are sophisticated and will negotiate. Expect:
- Multi-year commitment discussions (3-year is common)
- Volume discounts for large seat counts
- Annual payment (they prefer for budgeting)

Standard discounting: Up to 15% for 3-year, 10% for 2-year.

## Red Flags
- Early-stage startups with <$5M funding (churn risk)
- Cryptocurrency companies (regulatory uncertainty)
- Companies under investigation (compliance risk)
